# GBVC

This code is part of our new project titled "A Gradient-Based, Non-Expansive Visual Cryptography Scheme for High-Fidelity Reconstruction" which is ongoing review process in reputed journal.

1. Run the Python code (Process.py and ResAnal2.py) for processing.
2. Change the input file name, secret data file, and the output folder name in the program file.
3. Results will be saved in output folder (rename the folder name as per yours).
4. Authors used 'Taj.jpg' and 'Xray.jpg' as input image in this program.
5. For Dataset based pricessing, run the file, (dataset-plot.py) and use the folder 'dataset_images'
   (Add the dataset images inside the folder before processing)
